# Notes

This component is used to add new movies to the collection. It relies on NGRX Effects to get the data from the TMDB API.

## Actions

- Search
- SearchCompleted
- AddMovieToCollection
- MovieAddedToCollection

## Effects

- SearchMovies
- AddMovieToCollection
